# Humor — the routing map

Humor is not a mood you switch on; it is a **move routed to partner a feeling**. A
humor type that fights the routed feeling is betrayal by construction — gallows
humor at the wake works; slapstick at the wake is a defect (unless a character is
*choosing* it wrong, on purpose).

**Selection law:** read the dominant family's humor edge, then pick that partner
here. Plain word is canonical; the original name is the [preset skin](../presets/songcraft/README.md).

## The partner edges (the important table)

| Humor type | Preset | Partners the feeling | The move |
|---|---|---|---|
| **Dark / Gallows** | GallowsGrace | Grief + Anger | makes the unbearable bearable — honor, never cheapen |
| **Nervous** | AnxietyAlchemist | Fear | nervous energy made briefly manageable |
| **Teasing** | PlayfulPoke | Love | affectionate ribbing between people who are safe |
| **Banter** | BondingBanter | Solidarity | shared-experience laugh that builds a "we" |
| **Satire** | SatireSword | Solidarity / Anger | system critique through comedy |
| **Self-Deprecating** | DefensiveDeflector | Self-Compassion | protective comedy managing vulnerability |
| **Warm / Healing** | HealingGiggler | Self-Compassion / Calm | laughter as medicine; the gentle exhale-laugh |
| **Deadpan / Dry** | DeadpanDrafter | *against* Anger | dry understatement; the cut with more bite than volume |
| **Slapstick** | SlapstickSpark | Joy | physical comedy, chaos, delight |
| **Silly** | SillyLiberation | Joy | permission to be ridiculous |
| **Celebratory** | JestfulJoy | Joy | pure happiness amplified |
| **Wordplay / Wit** | WordplayWizard | (clever voices) | puns, linguistic play |
| **Absurd / Surreal** | AbsurdArchitect | (weird/dream logic) | non-sequitur, dream logic |
| **Observational** | StandUpForge | (any) | truth-telling by noticing out loud |
| **Parody** | ParodyPainter | (adapts to any) | comedic imitation with respect |
| **Roast** | RoastMaster | Anger — **handle with care** | aggressive/competitive; only constructive, never punching down |

## The harm lines (non-negotiable)
- No punching down. Ever.
- Aggressive spines (**Roast**) only used constructively — balance with a boundary.
- Dark humor **honors** what it processes. GallowsGrace makes the unbearable
  bearable; it never makes it cheap.

## The character filter
Humor lands in a *speaker's* voice. The same humor type sounds different in
different mouths — the type picks the **move**, the character's idiolect picks the
**words**. A humor move with no speaker attached is generic joking; route it to a
who.

## How your agent uses this
1. You already routed the dominant feeling in [`families/`](../families/).
2. Read that family's humor edge → find the partner row above.
3. Apply the **move**, in your speaker's voice, checked against the harm lines.
4. If humor would fight the routed feeling, use **none** — silence is a valid route.
