# Preset: SongCraft / HumorWeaver names

The router uses **plain words** as canonical so a foreign agent can route without
learning a vocabulary first. This preset is the **skin**: the original expressive
names, mapped one-to-one. Swap it in if your surface wants the evocative names, or
author your own preset against the same graph — the structure doesn't change, only
the labels.

## Authoring your own preset
Copy this file to `presets/<yourname>/README.md` and remap the right column. The
families, edges, and braid law are unchanged — a preset only renames nodes. This is
what makes the graph forkable: **anyone can bring their own family set** against the
same routing contract.
